**SQL Analytics: Digital Management & E-Commerce**

📊 **Project Overview**
This repository contains a collection of SQL scripts designed to solve real-world business problems. I focused on building structured databases and writing optimized queries to extract actionable insights from raw data, specifically focusing on inventory tracking and customer behavior.

🚀 **Key Technical Features**
Database Schema Design: Engineered relational table structures with Primary and Foreign Key constraints to ensure data integrity.
Advanced Querying: implemented complex logic using Aggregate Functions, Subqueries, and CASE Statements for data segmentation.
Business Logic: Developed scripts to calculate KPIs such as Average Order Value (AOV) and Loan Durations for operational auditing.
Data Cleaning: Used SQL to handle NULL values and format dates for time-series reporting.

📁 **Projects Included**
**1.** Digital Management System
Goal: Auditing a digital library/asset system to track borrowing trends.
Analysis: Identified "Power Borrowers" and categorized loan statuses to manage penalties and inventory.

**2.** E-Commerce Transaction Analysis
Goal: Analyzing customer spending patterns across different geographic regions (Mumbai, Pune).
Analysis: Created VIP customer lists by filtering users whose spending exceeded the platform average.


🛠️ **Tools Used**
SQL (MySQL): Relational Database Management.
Data Modeling: Entity-Relationship (ER) logic.
Analytical Thinking: Transforming raw rows into business summaries.
